# 트러블슈팅 로그

학습 도구를 쓰면서 발생한 문제와 해결 방법을 기록한다.

## 템플릿

```
### 문제: {한 줄 요약}
- **상황**: 뭘 하다가 발생했는지
- **증상**: 어떻게 잘못됐는지
- **원인**: 왜 그랬는지
- **해결**: 어떻게 고쳤는지
```

---

### 문제: 질문이 한 번에 3개씩 나옴
- **상황**: /start 1 실행 후 되묻기 단계
- **증상**: interface vs type, User[] vs Array<User>, optional property 질문이 한꺼번에 출력
- **원인**: CLAUDE.md에 "한 번에 1개만"이라는 제약이 없었음
- **해결**: CLAUDE.md에 "질문은 반드시 한 번에 1개만 던진다" 규칙 추가
